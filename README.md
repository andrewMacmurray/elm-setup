# Elm Setup

Frontend setup with Elm, Typescript, Parcel and Cypress

# Get up and running

Install dependencies

```sh
> npm install
```

Run the dev server

```
> npm run dev
```

Build the project

```
> npm run build
```

Run unit tests

```
> npm run test:unit
```

Run integration tests (run the dev server before running this)

```
> npm run test:integration
```

Run Cypress in interactive mode

```
> npm run cypress
```
